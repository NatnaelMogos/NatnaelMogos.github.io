Abstract

Hartree-Fock self-consistent field (SCF) calculations are the foundation of ab initio wave mechanic methods. Hence, the computation time of wave function methods depends on the convergence time of the SCF algorithm. In this work, we propose a workflow to learn the Hartree-Fock molecular orbital coefficients (MO) using high-dimensional neural network potentials (HDNNP).

Here, the HDNNPs map the Cartesian coordinates of a molecule to the corresponding MO matrix. The environment of each atom is encoded using so-called symmetry functions. We use these HDNNPs to predict initial guesses for the SCF routine to reduce the compu tational cost for the SCF algorithm.

We introduce a modification to the HDNNP architecture which enables us to reduce the number of trainable parameters. The modified HDNNPs map the input information to the atomic orbital coefficients (AO), which are then used in a second step to construct the MO matrix.
